# 🎬 Movie Recommendation System

A simple **Content-Based Movie Recommendation System** built using Python and Machine Learning. The system recommends similar movies based on genres, cast, crew, keywords, and movie overview.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* CountVectorizer
* Cosine Similarity
* Google Colab

## Dataset

* TMDB 5000 Movies Dataset
* TMDB 5000 Credits Dataset
